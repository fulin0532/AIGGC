# Roguelike Shooter - 肉鸽射击游戏

一款俯视角肉鸽射击游戏，参考《吸血鬼幸存者》和《次神:光之觉醒》的玩法机制，采用像素风格美术，快节奏的Roguelike体验。

## 🎮 游戏特色

- **俯视角射击**: 经典的上帝视角，清晰的战场视野
- **自动瞄准**: WASD移动，自动向最近的敌人射击，专注于走位策略
- **随机升级**: 升级时从3个随机能力中选择，每局体验不同
- **波次挑战**: 敌人波次生成，难度渐进式提升
- **Boss战**: 每5波出现强力Boss，考验玩家实力
- **角色成长**: 经验值系统，升级获得更强能力
- **本地存档**: 记录最高分和已解锁武器

## 🛠️ 技术栈

- **游戏引擎**: Phaser 3
- **构建工具**: Vite
- **渲染方式**: HTML5 Canvas
- **数据存储**: LocalStorage（纯前端，无需后端）

## 📦 游戏系统

### 核心玩法
- **移动**: WASD键控制角色移动
- **射击**: 自动向最近的敌人射击
- **收集经验**: 击杀敌人掉落经验宝石
- **升级选择**: 升级时选择能力提升

### 武器系统
1. **手枪** - 平衡的伤害和射速
2. **步枪** - 高射速，中等伤害
3. **霰弹枪** - 近距离高伤害，一次发射多颗弹丸

### 敌人类型（球球大军！）
1. **僵尸球球** 🔴 - 红色圆球，移动缓慢，带可爱表情
2. **奔跑者球球** 🟠 - 橙色圆球，速度快，兴奋笑脸
3. **坦克球球** 🔴⚫ - 深红色大球，血量高伤害高，凶狠表情
4. **Boss魔法球** 🟣✨ - 巨大紫色魔法球，带光环和王冠，每5波出现

### 升级选项（15种）

#### 基础升级
- **💊 生命提升** - 最大生命值+20
- **💊💊 大生命提升** - 最大生命值+50
- **⚡ 速度提升** - 移动速度+15%
- **⚡⚡ 大速度提升** - 移动速度+30%
- **⚔️ 伤害提升** - 攻击力+20%
- **⚔️⚔️ 大伤害提升** - 攻击力+50%
- **🔥 攻速提升** - 攻击速度+25%
- **🔥🔥 大攻速提升** - 攻击速度+50%

#### 进阶升级
- **🎯 射程提升** - 武器射程+30%
- **🚀 弹速提升** - 子弹速度+40%
- **❤️ 立即治疗** - 恢复50点生命值
- **✨ 经验加成** - 经验获取+50%
- **💥 暴击几率** - 15%几率造成双倍伤害
- **🩸 生命偷取** - 击杀敌人恢复5点生命
- **🔫 新武器** - 解锁一把新武器

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装依赖
```bash
cd roguelike-shooter
npm install
```

### 开发模式
```bash
npm run dev
```

游戏将在 http://localhost:8080 启动（如果端口被占用会自动切换）

### 生产构建
```bash
npm run build
```

构建产物在 `dist` 目录

### 预览构建
```bash
npm run preview
```

## 🎨 美术风格

目前使用像素风格图形：
- **玩家**: 可爱的像素柯基（棕色身体、白色肚子、短腿、小尾巴）🐕
  - 移动时会上下跳动（超级可爱！）
  - 短腿跑动动画
- **敌人**: 各种颜色的球球（柯基最爱玩球球！）
  - 🔴 **僵尸球球** - 红色，慢悠悠，圆眼睛
  - 🟠 **奔跑者球球** - 橙色，快速跳，笑脸表情
  - 🔴⚫ **坦克球球** - 深红色大球，凶狠脸
  - 🟣✨ **Boss魔法球** - 紫色带光环，有王冠
  - 每个球球都有高光效果和可爱表情
- **子弹**: 黄色圆点（暴击时显示红色并放大）
- **经验宝石**: 青色圆点（带闪烁效果）

**敌人显示**：
- 血条（颜色根据血量变化：绿→黄→红）
- 血量数字（当前/最大，例如：30/100）

**游戏主题**：柯基战士 VS 球球大军！所有球球都是柯基的玩具！🎾

后续可以替换为更精细的像素风格精灵图。

## 🔧 配置说明

游戏配置在 `src/config/GameConfig.js` 中，可以调整：
- 玩家属性（速度、血量、攻击力）
- 武器参数（伤害、射速、射程）
- 敌人属性（血量、速度、伤害）
- 波次配置（敌人数量、难度增长）
- 升级系统（经验值需求、升级效果）

## 📝 开发记录

项目使用 Phaser 3 游戏引擎从零开发，实现了完整的肉鸽射击玩法。

主要文件结构：
```
roguelike-shooter/
├── src/
│   ├── main.js                 # 游戏入口
│   ├── config/
│   │   └── GameConfig.js       # 游戏配置
│   ├── scenes/
│   │   ├── BootScene.js        # 启动场景
│   │   ├── MenuScene.js        # 菜单场景
│   │   ├── GameScene.js        # 游戏场景
│   │   └── GameOverScene.js    # 结束场景
│   └── entities/
│       ├── Player.js           # 玩家类
│       ├── Enemy.js            # 敌人类
│       ├── Bullet.js           # 子弹类
│       └── XPGem.js            # 经验宝石类
├── public/
│   └── assets/                 # 游戏资源
├── index.html                  # HTML入口
├── vite.config.js              # Vite配置
└── package.json                # 项目配置
```
